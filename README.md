# 2D Rocket League Clone

A simplified, top-down, browser-based 2D car soccer game built entirely with HTML5 Canvas and Vanilla JavaScript. 

## Features
* Custom collision and momentum physics engine.
* Momentum transfer (faster driving = harder ball strikes).
* Responsive keyboard controls.
* Zero dependencies (no external libraries required).

## How to Run the Game
1. Create a new folder on your computer named `rocket-league-2d`.
2. Save the provided `index.html` file into this folder.
3. Save the provided `style.css` file into this folder.
4. Save the provided `game.js` file into this folder.
5. Double-click `index.html` to open it in any modern web browser (Chrome, Firefox, Safari, Edge).
6. Install Python Editor for CMD.
7. Type, cd 'Where the folder is'
8. Then 'python -m http.server 8000'
9. Finally, go to 'http://localhost:8000'

## Controls
* **Up Arrow**: Accelerate forward
* **Down Arrow**: Brake / Reverse
* **Left Arrow**: Turn Left
* **Right Arrow**: Turn Right

## How to Play
You play as the Blue car. Your objective is to drive your car into the grey ball, using momentum to push the ball into the Orange goal (the right side of the screen). If the ball enters a goal, the score will update and the field will reset.
